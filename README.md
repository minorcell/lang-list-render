# 长列表渲染性能优化台

> 使用vue3 + ts + vite + pinia + vue-router + tailwindcss 实现。

## 主要包括：

1. 直接渲染十万条数据
2. 时间切片渲染

- 使用setTimeout模拟异步渲染
- 使用requestAnimationFrame模拟同步渲染

3. 虚拟列表渲染
   ...更多内容正在开发，目标是最终实现大多数前端性能优化场景的模拟。

## 使用

1. 安装依赖

```bash
pnpm install
```

2. 启动项目

```bash
pnpm run dev
```
